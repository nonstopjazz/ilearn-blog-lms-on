# Student Avatar Image

## 放置學生頭像圖片

請將學生頭像圖片命名為 `student-avatar.png` 並放置在此資料夾中。

### 圖片要求

- **檔名**: `student-avatar.png`
- **格式**: PNG, JPG, JPEG, WebP 均可（程式碼中使用 .png）
- **建議尺寸**: 至少 200x200 像素（會顯示為 96x96）
- **比例**: 正方形 (1:1) 最佳

### 如何添加圖片

1. 從 Google Drive 或其他來源下載圖片
2. 重命名為 `student-avatar.png`
3. 放置在 `/public/images/` 資料夾中
4. 重新部署應用程式

### 備用方案

如果圖片無法載入，系統會自動顯示預設的用戶圖標（藍色圓圈）。

### Google Drive 圖片連結

原始圖片來源：https://drive.google.com/file/d/1A9nnmElEb3sdho0sAuFDoLJ_dZg-_Rjf/view?usp=sharing

**下載步驟**：
1. 開啟上述連結
2. 點擊右上角「下載」按鈕
3. 將下載的圖片重命名為 `student-avatar.png`
4. 放置到此資料夾
